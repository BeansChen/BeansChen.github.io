基本运算符

赋值运算符

在C里面，符号 = 不表示“相等”，而是表示赋值，例如num=2019，不应该读成“num等于2019”，而应该读为把值2019赋给变量num，赋值运算符的动作是从右到左。

实际上，赋值运算符的左边必须指向一个储存位置。像2019=num这个是没有意义的。更普遍的，C中用“可修改的左值”来表示那些可以为之赋值的实体。

几个术语：数据对象，左值，右值，操作数。

“数据对象“(data object)是泛指数据存储区的术语，数据存储区能用于保存值。例如，用于保存变量或数组的数据存储区是一个数据对象。C的术语左值(lvalue)指用于标识一个特定的数据对象的名字或表达式。例如，变量的名字是一个左值。所以对象指的是实际的数据存储，但是左值是用于识别或定位那个存储的标识符。

因为不是所有的对象都是可更改值的，所以C使用术语”可修改的左值“来表示那些可以被更改的对象。所以，赋值运算符的左值应该是一个可修改的左值。

我们可以把变量理解为一个容器，放在左边当左值时，意思为保存、存放右边的值。所以，我们可以说=运算符的左操作数是可修改的左值。

数学运算符

单目运算符

只要一个操作数，自增++，自减–

双目运算符

需要两个操作数，+ - * / 和求模%

#include <stdio.h>

int main(){
    int a = 20,
        b = 10,
        c = 3;
    printf("a++=%d\n",a++);//20
    printf("++a=%d\n",++a);//22
    printf("a+b=%d\n",a+b);//32
    printf("a-b=%d\n",a-b);//12
    printf("a*c=%d\n",a*c);//66
    printf("a/c=%d\n",a/c);//7
    printf("a%%c=%d\n",a%c);//1

    return 0;
}

其他运算符

常年被人误认为函数的关键字”sizeof”

sizeof是C语言的32个关键字之一，并非“函数”，也叫长度(求字节)运算符,sizeof是一种单目运算符，以字节为单位返回某操作数的大小，用来求某一类型变量的长度。其运算对象可以是任何数据类型或变量。
例子：

#include<stdio.h>
int main()
{
        int n=0;
        int intsize = sizeof(int);
        printf("int sizeof is %d bytes\n",intsize);//int sizeof is 4 bytes
        return 0;
}

表达式和语句

表达式

表达式由操作数和运算符有构成，每个表达式都有一个整体值例如x = 1+2,整体值为3

语句

一个分号结尾代表一条语句

1;
1+1;
