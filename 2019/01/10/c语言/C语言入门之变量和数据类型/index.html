变量的基本概念

int a;//在内存中找了一块儿区域，用来存放整数，把它叫做a
a=20;//赋值，把整数20放在内存中
a=30;//重新赋值为30

可以从上面的代码看出，a并不是一成不变的，所以a是一个变量。

标识符

程序中的变量名，函数名，标号等称为标识符，除了函数库内置的函数，其余皆是用户自己来定义，
标识符由下划线_，数字0-9，英文A-z组成，且不能用数字开头，也不能和关键字重名。标识符几个注意事项：

  1. 标识符没有长度限制，但受到机器的限制，有可能只能识别一定位数的字符，比如只能识别8位，那么selfblog和selfblogchen就是一样的。
  2. 标识符是有大小写区分的。
  3. 尽量使标识符命名有意义，这样避免不必要的注释。

数据类型

内存中的数据有多种解释方式，使用之前必须要确定；上面的int a;就表明，这份数据是整数，不能理解为像素、声音等。int 有一个专业的称呼，叫做数据类型

小结

数据是放在内存中的，在内存中存取数据要明确三件事情：数据存储在哪里、数据的长度以及数据的处理方式。

变量名不仅仅是为数据起了一个好记的名字，还告诉我们数据存储在哪里，使用数据时，只要提供变量名即可；而数据类型则指明了数据的长度和处理方式。所以诸如int n;、char c;、float money;这样的形式就确定了数据在内存中的所有要素。

强类型和弱类型语言区别在于，强类型语言声明变量要声明类型，而弱类型语言不用，编译器会根据赋值情况自动推演出数据类型。
