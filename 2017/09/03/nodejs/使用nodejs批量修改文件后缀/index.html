使用nodejs批量修改文件后缀

2017-09-03
  • nodejs

前端在拿到新项目的原型图的时候，一般会先根据原型图搭好项目的目录结构，在进行多页面开发的时候，我们一般会把文件分为css，js，html的目录结构，比如下面这样：

通常 js 文件夹和 views 文件夹，也会有相同的文件结构，它们之间不同的只是后缀名，如果能有一个方法可以改后缀名，那么写一套另外两套也就高了，能省去一些重复性工作

// 使用方法为：
// 把该文件复制到要修改文件名的目录下；
// 然后打开node.js执行node changeName.js命令；
// 修改完成！
// 可以在下面注释的地方自定义修改的逻辑；
var path = require("path");
var fs = require("fs");
var rootPath = __filename;
renameFilesInDir(path.dirname(rootPath));

function changeFileName(filepath){
    fs.stat(filepath,function(err,stats){
        if(stats.isFile()){
            var filename = path.basename(filepath);
            var parentDir =path.dirname(filepath);
            var thisFilename = path.basename(__filename);
            //这个if就是进行更改文件名的逻辑,可以自行定义,这里定义为更改文件名称后缀"
            if(filename!=thisFilename){
                var newName = filename.replace(/html$/,"js");
                var newPath =parentDir+"\\"+newName;
                console.log("改变文件 "+filepath+" 为 "+newPath);
                fs.rename(filepath,newPath);    
            }
        }else if(stats.isDirectory()){
            console.log("=======["+filepath+"] 是一个文件夹=====");
            renameFilesInDir(filepath);
        }else{
        console.log("未知文件类型");
        }
    });
}

function renameFilesInDir(dir){
    fs.readdir(dir,function(error,files){
        var len = files.length;
        var file = null;
        for(var i=0;i<len;i++){
            file = files[i];
            changeFileName(dir+"\\"+file);
        }
    });
}

重复的机械工作会抹掉我们的积极性，遇到这种情况应该想办法让它变得简单。
