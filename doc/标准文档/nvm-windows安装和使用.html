日常开发中，经常会遇到手上不用的项目依赖不同版本NodeJS环境的情况，这就需要一个工具来管理不同版本NodeJS，官方推荐使用 nvm-windows (opens new window)。

# 安装

首先最重要的是：一定要卸载已安装的 NodeJS，否则会发生冲突。然后下载 nvm-windows (opens new window) 最新安装包（nvm-setup.zip），直接安装即可。

# 指定淘宝镜像

安装目录下找到settings.txt:

root: D:\nvm
path: D:\nvm\nodejs
node_mirror: https://npm.taobao.org/mirrors/node/
npm_mirror: https://npm.taobao.org/mirrors/npm/

或者命令行：

nvm node_mirror  https://npm.taobao.org/mirrors/node/
nvm npm_mirror  https://npm.taobao.org/mirrors/npm/

# 常用命令

nvm version		查看版本，说明安装已成功
nvm ls    列出已安装版本
nvm ls available   列出远程服务器上所有的可用版本
nvm use 4.2.2   使用指定版本
nvm install 4.2.2   安装指定版本
nvm uninstall 4.2.2  删除指定版本

nvm root D:\software\nvm\nodejs	 设置不同版本nodejs储存目录

# 可能会遇到问题

# nvm use [version] 报错“exit status 1 乱码”

导致该问题的可能原因有

1.安装路径有中文

2.安装路径有空格

3.cmd窗口要用管理员打开
