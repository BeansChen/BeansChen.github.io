# 前言

最近要开发一个飞书小程序，需要扩展小程序平台，但是在全局文件 (opens new window)配置中有提到：

扩展小程序平台时只能基于指定的基准平台扩展子平台，不能扩展基准平台。

# 配置

所以我们需要添加自定义平台，在package.json里添加配置。

 "uni-app": {
    "scripts": {
      "mp-feishu-dev": {
        "title":"飞书dev环境",
        "env": {
          "UNI_PLATFORM": "mp-toutiao"
        },
        "define": {
          "MAP-FEISHU-DEV": true
        }
      }
    }
  }

这样，就可以在代码中使用MAP-FEISHU-DEV来进行条件编译了

// #ifdef MP
小程序平台通用代码
// #endif
// #ifdef MP-TOUTIAO
字节平台通用代码
// #endif
// #ifdef MAP-FEISHU-DEV
飞书平台开发环境特有代码
// #endif

# 运行

打开hbuilderx，发现运行菜单下已经有了自定义平台

点击运行 飞书dev环境，会编译小程序到unpackage文件夹

注意飞书程序的编译目录依然是mp-toutiao,这时候就可以打开飞书开发者工具，打开对应目录来预览项目了。
