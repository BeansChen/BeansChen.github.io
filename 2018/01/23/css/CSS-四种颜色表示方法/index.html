CSS-四种颜色表示方法

2018-01-23
  • CSS

1.颜色名表示

例如：red、gold、yellow、green、blue、black、white等

2.16进制表示

即#RRGGBB 或 #RGB

例如：#333、#666、#8B8B8B等，注意:#F00全称是#FF0000而不是#F00F00

3.rgba表示

即red、green、blue、alpha，正整数值的取值范围为：0 - 255。百分数值的取值范围为：0.0% - 100.0%。超出会自动取极值

transparent是全透明黑色(black)的速记法，即一个类似rgba(0,0,0,0)这样的值

4.HSLA表示

  • H：Hue(色调)。0(或360)表示红色，120表示绿色，240表示蓝色，也可取其他数值来指定颜色。取值为：0 - 360
  • S：Saturation(饱和度)。取值为：0.0% - 100.0%
  • L：Lightness(亮度)。取值为：0.0% - 100.0%
  • A：Alpha透明度。取值0~1之间。
