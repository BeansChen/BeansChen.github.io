CSS多种边框的实现方式

2017-11-18
  • CSS

普通的边框

普通边框一般用border或者outline属性来实现，这没有什么好说的，要注意的
就只有outline是不占元素空间的

半透明边框

背景色默认的剪裁区域是border-box,要定义透明边框，不仅要设置边框透明属性，还要改变背景的剪裁范围

background: blue;
border:10px solid rgba(0,0,0,.3);
background-clip: padding-box;

多重边框

box-shadow: 0 0 0 10px red,0 0 0 20px yellow,0 0 0 30px blue;

图片边框

border: 40px solid transparent;
background: linear-gradient(white,white),url(2.jpg);
background-size: cover;
background-clip: padding-box,border-box;
background-origin: border-box;

边框内圆角

border-radius: 10px;
outline: 10px solid red;
box-shadow: 0 0 0 10px red;
