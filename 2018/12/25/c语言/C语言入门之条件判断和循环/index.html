条件判断语句

if else选择程序结构

形式一：

if(表达式){
    语句
}

形式二：

if(表达式){
    语句
}else{
    语句
}

形式三：

if(表达式1){
    语句1
} else if(表达式2){
    语句2
}else if(表达式n){
    语句.....
}else{
    语句
}

switch结构

switch给定一个表达式，当满足某个条件，执行对应的语句，如果遇到break语句则终止，否则继续往下执行。

#include <stdio.h>

int main(){
    int n;
    scanf("%d",&n);
    switch(n){
        case 1:
            printf("1");
            break;
        case 2:
            printf("2");
        case 3:
            printf("3");
        case 4:
            printf("4");
            break;
        default:
            printf("x"); 
    }

    return 0;
}

循环

C语言提供了三种循环结构，分别是for循环，while循环，do while循环，其形式如下：

while循环

while创建一个循环，当表达式成立为假之前重复执行，一旦条件不成立则终止循环

while(表达式){
    语句;
}

do while循环

和while循环区别在于先执行再判断，也就是至少会执行一次，无论表达式成立于否

for循环

for语句使用由分号隔开的三个控制表达式来控制循环过程。初始化表达式只在开始执行循环语句之前执行一次。如果判断表达式为真(或非0)就执行一次循环。然后计算更新表达式并再次检查判断表达式的值。for语句是一个入口条件循环，在进行一次循环之前决定是否要执行循环，因此有可能循环一次也不执行。循环的语句部分可以是一个简单的语句或一个复合语句。 

for(初始化表达式;判断表达式;更新表达式) 
{ 
循环体语句 
}

循环在判断表达式为假(或0)之前重复执行。

总结

循环和条件判断之前都可以嵌套，来实现复杂的循环判断需求。

循环体之间的选择：在知道循环什么时候结束的时候用for循环，否则用while循环
