2017-11-21
  • CSS

border实现各种形状

border实现三角形

width: 0;
height: 0;
border-width: 100px 200px 0 0;/*border画三角形*/
border-style: solid;
border-color: red transparent transparent transparent;

根据下面这个例子很容易得到我们想要什么样的三角形

width: 0;
height: 0;
border-width: 20px 40px;/*border画三角形*/
border-style: solid;
border-color: red gold blue black;

border实现梯形

width: 100px;
height: 0;
border-width: 0 100px 100px 200px;
border-style: solid;
border-color: transparent transparent red transparent;

border画椭圆

border单独指定水平和垂直半径， 只要用一个斜杠（ /） 分隔这两个值即可;border-radius: 50%/100% 100% 0 0;其实相当于border-radius: 50% 50% 50% 50% /100% 100% 0 0;

border-top-left-radius: 50% 100%;
border-top-right-radius: 50% 100%;
border-bottom-right-radius: 50% 0;
border-bottom-left-radius: 50% 0;

例子：

width: 0;
height: 0;
border-width: 200px 100px 0 100px;
border-style: solid;
border-color:red red transparent red; 
background: blue;
border-radius: 50%/100% 100% 0 0;

border实现左右等高布局

.fa{
        border-left:300px solid #222;
        display: inline-block;
       background: gray;
}
.son1{
    width: 300px;
    margin-left: -300px;
    float: left;
}
.son1 ul li{
    color: white;
}
.son2{
    float: left;
}
