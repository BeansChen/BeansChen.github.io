web常用几种图片格式的区别

2018-02-22
  • CSS

索引色最多256种，直接色理论上rgba256256256*256，但实际上不会有这么多

点阵图，最小单位像素，每个像素都有自己的信息，放大会失真

矢量图，用直线曲线和绘制方法组成，无论怎么放大，计算方式不变，所以不会失真

  • gif:无损压缩，支持透明和动画，体积小，只支持8bit的索引色，即256种不同颜色，适用于色彩要求不高且体积较小的场景，一般做一些动画小图标

  • jpeg：有损压缩，在肉眼不能分辨的情况下尽可能的压缩，使用直接色，色彩丰富，且可以颜色渐变，适用于色彩丰富的商品图，照片等

  • png-8：无损压缩，png的索引色版本，是gif的替代者，比gif小，且支持透明度调整，适用于小图标

  • png-24:无损压缩，直接色，体积较大，不支持透明

  • png-32:无损压缩，直接色，体积较大，支持透明

  • svg:无损的矢量图，可以编辑
